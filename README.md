# KMZ Overlay Editor（ハンターマップ調整ツール）

狩猟区域などの KMZ/KML ファイルを Mapbox 地図上に重ねて表示し、4コーナーを自由に調整して正確な位置に合わせたうえで KMZ として書き出すためのブラウザ完結型ツールです。

---

## 必要なもの

- **Mapbox アクセストークン**（無料プランで取得可能）
- モダンブラウザ（Chrome / Edge / Firefox 推奨）
- インターネット接続（Mapbox 地図の読み込みに必要）

---

## セットアップ

1. `config.example.js` を `config.js` という名前でコピーします。

   ```
   cp config.example.js config.js
   ```

2. `config.js` をエディタで開き、Mapbox アクセストークンを設定します。

   ```js
   const CONFIG = {
     mapboxToken: 'YOUR_MAPBOX_ACCESS_TOKEN', // ← ここを書き換える
     tilesetId: '',         // 任意（後述）
     tilesetLayerName: '',  // 任意（後述）
   };
   ```

   トークンは [https://account.mapbox.com/access-tokens/](https://account.mapbox.com/access-tokens/) で取得できます。

3. `index.html` をブラウザで開きます（ローカルファイルとして直接開いて動作します）。

---

## 使い方

### 1. KMZ ファイルを読み込む

左サイドバーの「KMZをドロップ」エリアにファイルをドラッグ＆ドロップするか、クリックしてファイルを選択します。

- 対応形式：`.kmz`、`.kml`
- KMZ 内の画像（地図画像）と座標情報が自動で読み込まれ、地図上に重なって表示されます。
- 座標情報がない KML の場合は、大阪付近のデフォルト位置に配置されます。

### 2. 透明度を調整する

スライダーで画像の透明度（0〜100%）を変更できます。下の地図と見比べながら調整してください。

### 3. 位置を調整する

画面上部に 3 つのモードボタンがあります。

| モード | 操作内容 |
|---|---|
| **地図操作** | Mapbox 地図のスクロール・ズームが有効。画像は動かない |
| **オーバーレイ調整** | 画像の4コーナーを個別に赤いハンドルでドラッグして変形・回転できる。中央の大きいハンドルで全体を移動できる |
| **画像移動** | 画像全体を平行移動する |

座標（緯度・経度）は左サイドバーの「3. 座標調整」欄に数値で直接入力することもできます。入力後は「**適用**」ボタンを押してください（入力中もリアルタイムで反映されます）。

4コーナーの対応：

- **NW**（北西）：画像の左上
- **NE**（北東）：画像の右上
- **SW**（南西）：画像の左下
- **SE**（南東）：画像の右下

### 4. KMZ として書き出す

左サイドバーの「4. エクスポート」欄でファイル名を入力し、「**KMZとして書き出し**」ボタンを押すとダウンロードが始まります。

- 書き出される KMZ は `gx:LatLonQuad`（4コーナー座標）と `LatLonBox`（バウンディングボックス）の両方を含むため、Google Earth や QGIS など各種 GIS ソフトで読み込めます。

「**現在の画像にズーム**」ボタンで、画像が画面に収まるよう地図ビューをリセットできます。

## Mapbox タイルセットの重ね表示（任意）

`config.js` の `tilesetId` と `tilesetLayerName` を設定すると、Mapbox Studio でアップロードした独自タイルセット（ポリゴン・ラインなど）を地図上にあらかじめ重ねて表示できます。
例えば、国土数値情報の鳥獣保護区のGeoJsonをMapboxにアップロードすれば、過去の保護区を目印に地図を移動させて精密な位置合わせが可能です。

```js
tilesetId: 'username.xxxxxxxxxx',   // Mapbox Studio のタイルセット ID
tilesetLayerName: 'source-layer-name', // ソースレイヤー名
```

不要な場合は空文字 `''` のままにしてください。

```

config.example.js   config.js のテンプレート
```
